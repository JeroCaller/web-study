JavaScript is one of the widely used programming language in web development and used to make web page dynamic and interactive. HTML and CSS mainly focus on defining the structure and design of web page, but JavaScript plays a role in making web pages more lively. JavaScript is a script language that is executed directly on client's web browser. It allows users to interact with web pages and it provides a lot of functions such as event handling, dynamic contents update, and animation. Besides, JavaScript is also used by server. In this case, you can develop server's application using runtime environment like Node.js. JavaScript shapes rich ecosystem to be used by both frontend and backend in entire web development stack. JavaScript has main functions and properties as follows:

  1. Event handling: it can control the behavior of web page by responding to interaction of users.
  2. Asynchronization programming: it improves web pages' performance through the asynchronous communication with server by AJAX and other technology.
  3. DOM manipulation: it can manipulate DOM(document object model) by Javascript to turn web pages and contents to be dynamic.
  4. Animation and graphic: It can draw graphic elements on web pages and create animations by technology such as Canvas and SVG.
  5. Modularization: JavaScript provides module system to divide codes into reusable modules.
JavaScript is a important language in modern software development as it is used in various fields such as mobile application, game, server-side application, as well as web.
